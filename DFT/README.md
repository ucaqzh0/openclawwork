# xiaozhua — DFT 自动化流程

## 概述

本库为 openclaw 自动化流程的 DFT 部分。**所有流程**均支持：

- **任务拆分**：各 tool 可独立执行
- **任务清单**：标准模板，发给 clio 或本机执行
- **移交检查**：提交/移交前必检
- **双模式**：xiaozhua（服务器）与 clio（功能机）均可执行

## 执行模式

### xiaozhua 执行

- 编辑对应流程的 `shared/config.py`，设置本机路径
- 直接运行各 tool 的 notebook

### clio 执行（发送执行）

- 从 [任务清单.md](任务清单.md) 进入对应流程的任务清单
- 填写后发给执行人
- 执行人在 clio 上：编辑 config（clio 路径）→ 做移交检查 → 执行勾选步骤

## 四要素说明

| 要素 | 说明 |
|------|------|
| **任务拆分** | tool_01, tool_02, ... 结构，每步可单独运行 |
| **任务清单** | 每流程有 `任务清单_流程名.md`，从 _template 复制后填写 |
| **移交检查** | 每流程有 pre_check 或 移交检查.md，提交前必执行 |
| **双模式** | config 统一配置，路径按 xiaozhua/clio 分别填写 |

详见 [流程开发指南.md](流程开发指南.md)。
