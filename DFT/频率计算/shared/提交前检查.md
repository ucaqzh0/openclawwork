# 频率计算 — 提交前必检

每次提交频率任务前，必须完成以下检查。

---

## 1. 原子固定

- **要求**：Cu 基底原子全部固定（Selective Dynamics F F F）
- **检查**：运行 `pre_check.py` 或人工核对 POSCAR
- **不通过**：禁止提交，先运行 `prepare_structure` 或手动修正

---

## 2. INCAR

- **要求**：各任务目录存在 INCAR，关键参数符合运行规则 2.4
- **关键项**：IBRION=5, ISIF=2, NSW=1, NFREE=2 等
- **检查**：`pre_check.py` 或人工核对

---

## 3. 额外修改

- **要求**：若有手动修改（如 ISPIN、EDIFF 等），需记录
- **记录位置**：任务清单 3.3 或本目录下 `extra_modifications.txt`

---

## 使用 pre_check

```bash
cd 频率计算/shared
python pre_check.py
```

或在 notebook 中：

```python
from shared.pre_check import run_pre_check
run_pre_check(zpe_root)
```
