# build — 移交检查

## 适用范围
用于把 `build` 阶段生成的“最基本结构”交接给后续 `tool_02/03/04`（或其它输入文件生成流程）。

## 1. 结构文件存在
- 每个结构目录必须至少存在一个 `.vasp` 结构文件
- 结构命名与目录层级需与 `@absorption` 的 `1_body/2_body` 分类规则一致

## 2. 原子顺序约定（NEB 之前的通用前置）
- 对于单吸附与双吸附：原子顺序必须与后续对齐/筛选逻辑兼容（至少保持“Cu 基底在前 + 吸附原子在后”的稳定约定）

## 3. 额外修改记录
- 若存在手动改动（例如人工替换 .vasp、额外旋转/平移等），需写明改动位置和原因
- 记录位置：任务清单第 3.3（或另建 `extra_modifications.txt`）

## 使用说明
- 本阶段不涉及 `nebmake.pl`，不要求 `ini/fin` 输出。
