# build — 流程映射

本目录用于把你已有的 GO/吸附最佳结构流程，**并行化为“结构建立（build）”**。

## tool 对应（建议输出只到 best 结构）
1. `tool_01_文档建立`：建立数据架构与目录约定（参照 `@absorption` 的 `1_body/2_body`）
2. `tool_02_单吸附构建`：生成单吸附基础结构；得到并导出单吸附 best 结构（用于后续双吸附）
3. `tool_03_双吸附构建`：根据单吸附 best 推导/生成双吸附基础结构；用对齐策略最小化位移（避免优化后混乱）
4. `tool_04_提取与整理`：从 `1_body/Final`、`2_body/Final` 汇总到 `thermol/Cu_100_absorption`，并生成地址清单（用于直接索引与去重）
