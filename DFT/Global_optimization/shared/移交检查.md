# Global Optimization — 移交检查

每次提交 GO 任务前，必须完成以下检查。**返回检查单时需确认四类文件齐全**。

---

## 1. 输入文件完整（必检）

各任务目录（`job_*/`）必须具备：

| 文件 | 说明 |
|------|------|
| **POSCAR** | 结构文件（来自 .vasp 或手动准备） |
| **POTCAR** | vaspkit -task 103 生成，元素顺序与 POSCAR 一致 |
| **KPOINTS** | 模板或手动设置 |
| **INCAR** | 模板或手动设置 |

- **检查**：运行 `pre_check.py` 或人工核对
- **不通过**：禁止提交，补全后再提交

---

## 2. POTCAR 与 POSCAR 匹配

- **要求**：POTCAR 元素顺序与 POSCAR 严格一致
- **检查**：人工或脚本核对
